# Security Evaluation Experiment Log

This repository records public notes, experiment summaries, and data references for a Kaggle security benchmark competition.

## Scope

The repository is intentionally limited to public-facing material:

- project background and data notes
- experiment summaries and public metrics
- release notes for public artifacts
- sanitized result tables or figures when available

It does not contain credentials, private data, local runtime logs, submission packages, or internal automation details.

## Repository Layout

- `docs/` - public documentation and release boundaries
- `experiments/` - dated experiment records
- `public_data/` - public data references and schemas
- `results/` - sanitized summaries and figures

## Current Status

Preparation completed on 2026-06-13:

- competition access verified
- official files stored locally, outside git tracking
- local readiness checks completed
- public repository boundary established

## Data

Competition files are stored in an ignored local directory. Raw downloads and local caches are not committed. Public data notes are maintained in `public_data/`.
